# API バージョン 16 の新機能

Adobe Workfrontは、2022 年 4 月 7 日に API バージョン 16 をリリースしました。 API バージョン 16 では、バージョン 15 から次の変更がおこなわれました。

## 追加されたリソース

API バージョン 16 に関するリソースは追加されませんでした。

## 削除されたリソース

API バージョン 16 で削除されたリソースはありませんでした
