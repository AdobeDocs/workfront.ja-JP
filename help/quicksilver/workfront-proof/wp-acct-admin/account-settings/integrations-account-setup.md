# での統合用アカウント設定 [!DNL Workfront Proof]

>[!IMPORTANT]
>
>この記事では、スタンドアロン製品の機能について説明します [!DNL Workfront Proof]. 内部での検証に関する情報 [!DNL Adobe Workfront]を参照してください。 [校正](../../../review-and-approve-work/proofing/proofing.md).

As a [!DNL Workfront Proof] 管理者、 [!DNL Workfront Proof] と統合する [!DNL Basecamp], [!DNL Basecamp Classic], [!DNL NetSuite]、および [!DNL Workfront].

統合の一般的な接続を確立したら、各ユーザーは [!UICONTROL 統合] タブが個人設定ページに表示されます。

詳しくは、 [統合記事](https://support.workfront.com/hc/en-us/categories/115000588707-Integrations) 」を参照してください。
