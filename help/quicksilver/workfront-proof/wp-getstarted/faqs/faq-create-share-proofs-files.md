# FAQ — 配達確認とファイルの作成と共有

>[!IMPORTANT]
>
>この記事では、スタンドアロン製品の機能について説明します [!DNL Workfront Proof]. 内部での検証に関する情報 [!DNL Adobe Workfront]を参照してください。 [校正](../../../review-and-approve-work/proofing/proofing.md).

## 配達確認とは？

### 回答

配達確認は、静的、音声またはビデオのファイルで、校正ビューアで確認できます。 配達確認に追加されたレビュー担当者は、配達確認に対するコメントや決定をおこなうための一連のツールを使用できます。

## どのファイルタイプがサポートされていますか？

### 回答

配達確認は、静的、オーディオおよびビデオの各ファイルから作成できます。 4GB を超えるファイルはアップロードできません。 [!DNL Workfront] は 150 を超えるファイルタイプをサポートしています ( [サポートされている検証ファイルの種類とサイズ制限の概要](../../../review-and-approve-work/proofing/proofing-overview/supported-proofing-file-types.md) を参照 )。

## 配達確認とファイルの違いは何ですか？

### 回答

ファイルをにアップロードする際 [!DNL Workfront Proof]の場合、ファイルは [!DNL Workfront Proof] アカウント 共有する場合は、 [!DNL Workfront Proof] 受信者にリンクを電子メールで送信します。このリンクをクリックすると、ファイルをダウンロードできます。 任意の種類のファイルを共有できます。

ファイルから配達確認を作成する際に、アップロード先の [!DNL Workfront Proof]を使用すると、ファイルを校正ビューアでレビューできるようになります。 レビュー担当者に、配達確認へのリンクが記載された E メールが届きます。 配達確認を開くと、配達確認の画像を表示し、コメントを追加して、それに関する決定を下すことができます。 サポートされているファイルタイプのリストにあるファイルを使用して、配達確認を作成できます。 また、Web サイトやその他の Web コンテンツ用の URL を使用して配達確認を作成できます。

サポートされているファイルタイプの完全なリストについては、 [サポートされている検証ファイルの種類とサイズ制限の概要](../../../review-and-approve-work/proofing/proofing-overview/supported-proofing-file-types.md).

## 配達確認を作成するにはどうすればよいですか？

### 回答

静的ファイル、オーディオファイル、ビデオファイルおよび URL から配達確認を作成できます（を参照）。

アカウントで配達確認を作成するには、適切な権限プロファイルを持つユーザーである必要があります ( [[!UICONTROL 配達確認権限プロファイル] in [!DNL Workfront Proof]](../../../workfront-proof/wp-acct-admin/account-settings/proof-perm-profiles-in-wp.md)) をクリックします。

複数のファイルを同時にアップロードする場合は、1 つの E メールを使用して、同じレビュー担当者グループに送信できる複数の配達確認を作成します。 組織が [!UICONTROL 大規模法人] または [!UICONTROL 無制限] アカウントを使用すると、ファイルを 1 つの配達確認に組み合わせることができます ( [複数ページの配達確認の作成](../../../review-and-approve-work/proofing/creating-proofs-within-workfront/create-multi-page-proof.md)) をクリックします。

## 配達確認の役割と E メールのアラートとは何ですか？

### 回答

配達確認の役割は、レビュー担当者が配達確認に対しておこなう必要のあるアクションを定義します。 配達確認を作成する際に、レビュー担当者に対して使用できる役割のオプションは、コメントを作成したり、意思決定をおこなったりできるかどうかに応じて異なります。 詳しくは、 [での配達確認の役割の管理 [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/share-proofs-and-files/manage-proof-roles.md).

E メールアラートは、配達確認の進行状況をレビュー担当者に更新します（新しい配達確認および遅延配達確認通知とは異なります）。 配達確認での各レビュー担当者の役割に応じて、レビュー担当者ごとに異なるオプションを選択できます。 詳しくは、 [を使用した高度な配達確認の作成 [!UICONTROL 自動ワークフロー]](../../../review-and-approve-work/proofing/creating-proofs-within-workfront/create-automated-proof-workflow.md).

## 複数のファイルから配達確認を作成できますか？

### 回答

複数のファイルを 1 つの配達確認に組み合わせる機能は、 [!UICONTROL 大規模法人] および [!UICONTROL 無制限] エディションプラン このオプションは静的ファイルに対してのみ使用でき、ビデオファイルに対しては使用できません。 詳しくは、  [複数ページの配達確認の作成](../../../review-and-approve-work/proofing/creating-proofs-within-workfront/create-multi-page-proof.md).

## URL から配達確認を作成できますか？

### 回答

はい、Web サイトやその他の Web コンテンツから配達確認を作成できます。 配達確認を作成する URL を追加する際に、静的な配達確認とインタラクティブな配達確認のどちらにするかを指定できます。

* インタラクティブな配達確認では、レビュー担当者は、Web サイトや、ストリーミングビデオやオーディオを使用した広告などの他の Web コンテンツと通常通りに移動したりやり取りしたりできます。 [!DNL Flash] 広告、HTMLアニメーション、インタラクティブバナーの要素。 詳しくは、 [ZIP ファイル内のインタラクティブコンテンツの配達確認の作成](../../../review-and-approve-work/proofing/creating-proofs-within-workfront/generate-proof-interactive-content-.md).
* 静的配達確認の場合、 [!DNL Workfront] は、指定したページとサブページのスクリーンショットを取得します。 ハイパーリンクは配達確認内で有効になるので、正しい配達先に導くかどうかをテストできます。 詳しくは、 [Web サイトまたはその他の Web コンテンツ用の静的配達確認の作成](../../../review-and-approve-work/proofing/creating-proofs-within-workfront/generate-static-proof-website-other-web-content.md).

複数の URL をスペースで区切ると、複数の URL を一度に追加できます。 組み合わせは、 [!UICONTROL 大規模法人] および [!UICONTROL 無制限] エディションプラン

## ログイン情報を持たない人は、自分のアカウントで配達確認を作成できますか？

### 回答

## 配達確認の共有とは何ですか？

### 回答

配達確認を共有すると、レビュー担当者がその配達確認にアクセスして、コメントやマークアップを追加して、その配達確認に関する決定を下すことができます。 ゲストレビュー担当者は、受け取った電子メール通知から配達確認にアクセスします。 自分のレビュー担当者 [!DNL Workfront Proof] アカウントは、 [!UICONTROL ダッシュボード].

配達確認に対する十分な権限を持っている場合は、校正ビューアを使用できます。 [!UICONTROL ダッシュボード]または任意のリスト表示回数を使用して、既存の配達確認にレビュー担当者を追加できます ( [で配達確認を共有 [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/share-proofs-and-files/share-proof.md) and [Manage Items on the [!UICONTROL Views] ページイン [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-items-on-views-page.md).

## 配達確認を確認するには、ユーザーである必要がありますか？

### 回答

いいえ. ゲストレビュー担当者 ( [!DNL Workfront Proof] ログイン資格情報 ) は、受信した配達確認の電子メール通知から配達確認にアクセスできます。 配達確認は、必要な数のゲストと共有できます。

以下の条件を持つユーザーに対して、配達確認の共有を制限できます。 [!DNL Workfront Proof] ログイン資格情報。 これにより、配達確認にセキュリティのさらなる層が追加されます。 セキュリティ強化のため、 [!UICONTROL 大規模法人] および [!UICONTROL 無制限] プランでは、組織で作成されたすべての配達確認に対して、この要件を設定できます。

ログインの要求について詳しくは、 [での配達確認のセキュリティ [!DNL Workfront Proof]](../../../workfront-proof/wp-acct-admin/managing-security/proof-security-in-workfront-proof.md).

組織で、レビュー担当者が電子的に配達確認に署名する必要がある場合は、次のログインが必要です。 [!DNL Workfront Proof]を使用すると、ユーザーは登録済みユーザーとのみ配達確認を共有できます。 これは、で利用できます。 [!UICONTROL 大規模法人] および [!UICONTROL 無制限] エディションプラン 詳しくは、 [での電子署名について [!DNL Workfront Proof]](../../../workfront-proof/wp-acct-admin/managing-security/electronic-sigs-in-wp.md).

## レビュー担当者の期限を設定できますか？

### 回答

配達確認の作成時に、新しい配達確認の期限を設定したり、新しい配達確認のバージョンを設定したりできます。 これは、 [!UICONTROL ワークフロー] セクション [!UICONTROL 新しい配達確認] ページ。 次を使用する場合、 [!UICONTROL 自動ワークフロー]の場合は、レビューの各ステージに異なる期限を設定できます。

また、 [!UICONTROL 配達確認の詳細] ページ。 詳しくは、 [で配達確認の詳細を管理 [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-proof-details.md).

## 配達確認の新しいバージョンを作成するにはどうすればよいですか？

### 回答

レビュー担当者は、多くの場合、配達確認に対するコメントの変更を要求し、新しいバージョンの配達確認を表示したいと考えます。 配達確認の新しいバージョンを作成できます。 [!DNL Workfront Proof] では、以前のバージョンからの配達確認の設定が記憶されます。 配達確認のレビュー担当者の追加や削除などの作業が必要な場合は、これらの設定を編集できます。

各バージョンを、表示する必要のある特定のレビュー担当者と共有する必要があります。 例えば、1 人のレビュー担当者とバージョン 3 のみを共有している場合、そのユーザーにはバージョン 1 とバージョン 2 は表示されません。 アカウント内のスーパーバイザーと管理者は、アカウント内のすべてのプロジェクトを監視し、配達確認のすべてのバージョンを表示および編集できます。

詳しくは、を参照してください。

## 次を使用してファイルを共有できますか？ [!DNL Workfront Proof]?

### 回答

はい. 他のユーザーと何かを共有したいのに、配達確認として表示する必要がない場合（または、がサポートしていないファイルの種類の場合） [!DNL Workfront Proof]など )、ファイルとして [!DNL Workfront Proof] アカウント 配達確認と同様に、ファイルをフォルダーやタグファイルに整理し、ファイルを共有する際にカスタムメッセージを通知 E メールに追加できます。 詳しくは、 [ファイルと Web コンテンツのアップロード先 [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/create-proofs-and-files/upload-files-web-content.md).

共有しているファイルに関する電子メール通知を受け取った受信者は、通知内のリンクをクリックしてファイルをダウンロードできます。

[!DNL Workfront Proof] ユーザーは、アカウントに保存したファイルを配達確認に変換できます。
