# プルーフと Creative Cloud Express の概要


Proof と Creative Cloud Express の新しい統合を導入できるのを非常に誇りに思います。 この統合を使用すると、次のことができます

* Workfront承認テンプレートを使用した承認ワークフローの定義

* プルーフビューアでの Creative Cloud Express ドラフトコンテンツの確認

* 最終承認と公開方法については、Creative Cloud Express での決定の確認を参照してください

## 前提条件

* Workfrontと Creative Cloud Express は、同じ IMS 組織にデプロイする必要があります。

* 統合を機能させるために、IMS 組織内の 1 つのWorkfront インスタンスでのみ、ユーザーをアクティブにして有効にできます

* Workfront インスタンスは、統合シェルで有効にする必要があります

* Workfrontの設定セクションがプルーフに接続するように切り替えられました（以下を参照）
