# Workfront のカスタムフォームへの計画情報の追加

Workfrontのカスタムフォームに「Planning 接続」フィールドタイプを追加すると、Adobe Workfront Planning 接続レコードをWorkfront オブジェクトのカスタムフォームに表示できます。
