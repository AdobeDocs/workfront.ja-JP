# Workfront Planning用Workfront Data Connect

Workfront Data Connectでは、Snowflakeを利用して、Workfront計画からAdobe Workfrontのデータにアクセスし、ビジネスに不可欠な疑問に答え、プロセスの価値を測定できます。 エンタープライズ版のお客様は、Workfrontデータレイクから直接データビジュアライゼーションツールを使用してデータを分析したり、Workfrontデータを顧客所有のデータウェアハウスに書き出したりできます。

加入者は、選択したビジネスインテリジェンスツール（Tableau、Power BI、Domo、Lookerなど）を接続できます。 次のタイプのデータを使用してデータビジュアライゼーションを作成するには、Data Connectを実行します。

* 低遅延、現状データ
* 特定の時点を比較するためのイベントレコードの変更
* トレンド分析のための日々の履歴スナップショット

Snowflakeを使用してリーダーアカウントを作成し、Workfront Data Connectへの接続を確立する方法については、次の記事を参照してください。

* [Snowflake のリーダーアカウントまたは接続の作成](/help/quicksilver/reports-and-dashboards/data-lake/create-a-reader-account.md)
* [Workfront Data Connect への接続の確立](/help/quicksilver/reports-and-dashboards/data-lake/share-data-externally.md)


Workfront Data Connect データディクショナリのプランニング情報について詳しくは、[Workfront Data Connect データディクショナリ ](/help/quicksilver/reports-and-dashboards/data-lake/data-dictionary.md)を参照してください。
