# 場所の設定

{{highlighted-preview-article-level}}

レートカードを使用すると、組織でプロジェクトの請求率を簡単に管理できます。 詳しくは、 [レートカードの管理](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/manage-rate-cards.md). As a [!DNL Adobe] [!DNL Workfront] 管理者は、レート・カードでジョブ・ロールに割り当てる際に使用できるデフォルトの場所を設定できます。 これにより、レートカードは各場所の市場レートを正確に反映します。

## 組織全体の場所を設定する

{{step-1-to-setup}}

1. 左側のパネルで、 [!UICONTROL **場所**].
1. クリック [!UICONTROL **場所をさらに追加**].
1. 場所の名前と説明を入力します。

   場所名は大文字で始める必要があり、場所を重複させることはできません。

1. 入力領域の外側をクリックして、場所を保存します。
1. ロケーションを削除するには、リストでロケーションを選択し、 **削除** アイコン ![削除アイコン](assets/delete.png).

>[!NOTE]
>
>レートカード上のジョブロールに関連付けられているロケーションは削除できません。
