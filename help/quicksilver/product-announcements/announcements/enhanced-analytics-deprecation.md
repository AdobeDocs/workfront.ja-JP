# 拡張分析の廃止ガイド

使用の減少と機能の制限により、Enhanced Analyticsは間もなくAdobe Workfrontから非推奨（廃止予定）になります。 Workfront Data Connectは、Enhanced Analyticsでは不可能だった強力な新しいビジュアライゼーションの作成に加えて、現在使用しているEnhanced Analytics ビジュアライゼーションを複製できる代替手段としてリリースされました。 移行を成功させるために、このページでは、何が正確に変更されるのかに関する情報、移行の準備方法に関するアドバイス、よくある質問への回答を掲載しています。

## 非推奨（廃止予定）情報

**何が変更されますか？**

* Enhanced Analyticsは使用できなくなります。

**いつ？**

* 5月26日の週

**代わりに何を使用できますか？**

* Workfront Data Connectは、拡張分析ビジュアライゼーションをレプリケートできます。 詳しくは、以下の「[Workfront Data Connectでの高度なAnalytics ビジュアライゼーションのレプリケート」を参照してください。](#replicate-enhanced-analytics-visualizations-in-workfront-data-connect)

## 廃止に備える

Adobe AnalyticsからWorkfront Data Connectへの移行を容易にする推奨事項を以下に示します。

### Workfront Data Connectの使用を開始

最初にお勧めするのは、お客様の組織がWorkfront Data Connectにアクセスできるようにし、できるだけ早くその使用に慣れることです。 Data Connect機能とその使用方法について詳しくは、[Workfront Data Connectの概要](/help/quicksilver/reports-and-dashboards/data-lake/data-lake-overview.md)を参照してください。

Data Connectは、新しいUltimate プラン <!--, and can be purchased as an add-on to the new Select and Prime plans-->に含まれています。 Data Connectは、従来のWorkfront プランでは利用できません。 現在お使いの組織のプランとData Connectの購入方法については、Workfront アカウント担当者にお問い合わせください。

### 拡張分析ビジュアライゼーションのレプリケート

Workfront Data Connectを使用すると、多くの新しいビジュアライゼーションオプションに加えて、最も価値のあるEnhanced Analytics ビジュアライゼーションをレプリケートできます。

次の記事のクエリを使用して、拡張分析のクエリと同様のデータビジュアライゼーションを作成します。


* [KPI クエリ](/help/quicksilver/reports-and-dashboards/data-lake/enhanced-analytics-queries/kpi-queries.md)
* [フライトプランクエリ](/help/quicksilver/reports-and-dashboards/data-lake/enhanced-analytics-queries/flight-plan-queries.md)
* [プロジェクトアクティビティクエリ](/help/quicksilver/reports-and-dashboards/data-lake/enhanced-analytics-queries/project-activity-queries.md)
* [プロジェクトツリーマップクエリ](/help/quicksilver/reports-and-dashboards/data-lake/enhanced-analytics-queries/project-tree-map-queries.md)
* [チーム別アクティビティクエリ](/help/quicksilver/reports-and-dashboards/data-lake/enhanced-analytics-queries/team-queries.md)

### 助けを求める

移行中に問題が発生した場合は、アカウント担当者またはサポートチームにお問い合わせください。
