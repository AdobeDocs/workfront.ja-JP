# 2019.3 リリースアクティビティの概要

このページでは、2019.3 リリースに含まれる機能について説明します。このページに記載の機能は、2019年8月19日（PT）の週に実稼動環境で使用できるようになりました。

## すべてのユーザー向けの機能強化

2019.3 リリースには、次の機能強化が含まれています。

* [開くフィルターで、より多くの結果がイシューのリストに表示されるようになりました](../../../../product-announcements/product-releases/quarterly-release-archive/2019.3-release-activity/2019.3-project-enhancements.md#changes)
