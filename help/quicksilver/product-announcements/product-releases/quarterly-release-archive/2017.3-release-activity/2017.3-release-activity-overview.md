## モバイルの強化 {#mobile-enhancements}

>[!NOTE]
>
この節で説明する機能が、Workfront Production アカウントで使用できるようになりました。これは、プレビュー機能ではありません。

2017 年 8 月に、Android とiOSの両方のWorkfrontモバイルアプリの実稼動バージョンに次の機能が追加されました。

* リクエストを送信
* タイムシートの新規エントリ
* カスタムフォーム編集
* 配達確認の承認リクエスト
* 購読項目に対するプッシュ通知

これらの変更について詳しくは、 [iOSおよび Android 用のモバイルアプリを更新しました（2017 年 8 月初旬）](../../../announcements/announcement-archive/updated-mobile-app.md).

11 月 8 日の週に、実稼動環境のリリースより前に、次の追加機能が使用可能になります。

* プロジェクトのリストへのアクセス
* プロジェクト上のタスクとサブタスクのリストへのアクセス
* プロジェクトの問題のリストへのアクセス
* プロジェクトに新しい問題を記録する
