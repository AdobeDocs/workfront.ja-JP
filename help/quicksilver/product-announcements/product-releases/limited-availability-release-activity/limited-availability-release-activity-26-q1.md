# Adobe Workfront の限定提供機能に関する 2026年第 1 四半期のリリースアクティビティ

この記事では、2026年第1四半期中にリリースされるWorkfront機能の限定提供について説明します。

<!--keep the sentence below for all future quarterly release pages-->

Adobe Workfront向けにリリースされたすべての制限付き機能の一覧については、[Adobe Workfront制限付き機能リリースアクティビティ：記事インデックス &#x200B;](/help/quicksilver/product-announcements/product-releases/limited-availability-release-activity/limited-availability-release-activity-article-index.md)を参照してください。

## 制限付き機能のリリースノート

この節には、新機能のリリースノートが含まれています。

### レートカードでの担当業務の割合の調整

>[!NOTE]
>
>プロダクションリリース日：2025年11月20日（PT）

レートカード内の1つ以上の担当業務の請求レートにパーセンテージベースの変更を適用できるようになりました。 各レートを1つずつ手動で更新する代わりに、1回の操作で選択したすべてのレートに変更を適用できます。

### 高度な検索におけるKPIの制限

>[!NOTE]
>
>プロダクションリリース日：2025年12月18日（PT）

高度な割り当ておよび人員配置プランの高度な検索の使いやすさを向上させるために、テーブル列に表示できるKPIのリストが制限されるようになりました。 主要な指標のみを表示することで、情報をより迅速に見つけることができます。

* 高度な割り当て高度な検索では、**利用可能なキャパシティ** KPIを使用できます。 この指標は、ユーザーの基本合計容量を表します。
* スタッフ計画の詳細検索では、**残存スタッフ配分容量** KPIを使用できます。 この指標は、割り当て後も引き続き利用可能なキャパシティを反映します。

## 限定的な可用性の機能に関するメンテナンス更新

この節では、限定可用性の機能に関する週次Workfront アップデートで修正された問題について説明します。

### 2025年11月23 ～ 29日のメンテナンス更新週

**高度な割り当てに関する高度な検索から時間/FTE/割合が削除されました**

詳細割り当て詳細検索で、**時間 / FTE / パーセンテージ**&#x200B;のオプションを選択しても、検索表示が変更されませんでした。 ドロップダウンがツールバーから削除されました。

**タスク期間が0**&#x200B;の場合、詳細割り当てで予定時間を編集できません

タスク期間が0の場合、詳細割り当て画面の予定時間数KPIは編集できなくなります。 以前は、予定時間を編集することはできましたが、変更は保存されませんでした。

### 2025年11月30日～12月6日（PT）の週のメンテナンスアップデート

**日付に有効な為替レートが使用されていない場合、「為替レート日付」フィールドが非表示になります**

「**システムから日付有効な為替レートを使用**」チェックボックスが選択されていない場合、プロジェクトの詳細/財務領域の「**為替レート日**」フィールドが非表示になります。 以前は、プロジェクトを編集する際に、日付に有効な為替レートが有効になっていない場合でも、プロジェクトの詳細に日付フィールドが表示されていました。

### 2025年1月4日～10日（PT）の週のメンテナンスアップデート

**担当業務の割り当て率で、有効日にギャップが生じるようになりました**

ジョブロール率に関する有効な日付は、重複する日付は許可されていませんが、ギャップを許可するようになりました。 ギャップの間、請求料率の階層の他の領域は、タスクの収益タイプに基づいて請求率を決定するために使用されます。
