# 重要な作業アイテムの優先順位付け

「My Focus」列を使用して、作業の優先順位を付けることができます。 マイフォーカス列は個人の値であり、タスクまたは問題に設定された優先度には影響しません。

「優先度」には、割り当てられた作業項目が表示されます。 チームに割り当てられた作業項目を表示できません。

## 「My Focus」列で作業の優先順位を付ける

{{step1-to-priorities}}

1. 対象となる作業項目を探します。
1. **マイフォーカス**&#x200B;列で、次のいずれかのフォーカスレベルを選択します。

   | Focus | 説明 |
   |-----------|-------------|
   | **緊急** | 緊急の対応が必要な作業項目に対して必要です。 緊急の作業項目は何よりも優先され、できるだけ早く対処されるべきである。 |
   | **高** | 「高」は、緊急の作業が完了した後に対処する予定の重要な作業項目に対して使用されます。 |
   | **標準** | 通常は、緊急アイテムと優先度の高いアイテムが完了した後に作業する日常的な作業アイテムに使用されます。 これは、タスクとイシューのデフォルトです。 |
   | **低** | 「低」は、すぐに注意を払う必要がない作業項目を指し、優先度の高い作業項目がすべて完了するまで延期することができます。 |

   ![自分のフォーカス &#x200B;](assets/my-focus-060226.png)

   >[!TIP]
   >
   >フォーカスレベル別に作業をフィルタリングしてグループ化できます。
