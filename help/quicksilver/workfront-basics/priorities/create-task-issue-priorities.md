## タスクまたはイシューの優先度の作成

プロジェクトでタスクまたはイシューを作成できます。 「優先度」には、割り当てられた作業項目が表示されます。 チームに割り当てられた作業項目は、優先順位ワークリストに表示できません。

>[!NOTE]
>
>リクエストまたはコントリビューターライセンスをお持ちの場合は、優先度で直接リクエストを作成できます。

プロジェクトでタスクまたはイシューを作成するには：

{{step1-to-priorities}}

1. 右上隅の「**新規作成**」をクリックします。
   ![新しい（実稼動環境）を作成](assets/create-new--.png)
1. 次の情報を指定します。

   | フィールド | 手順 |
   |---------------|-------------|
   | 作業項目タイプ（オプション） | **タスク**&#x200B;または&#x200B;**問題**&#x200B;を作成するかどうかを選択します。 |
   | 名前 | 名前を入力してください。 |
   | 説明（オプション） | 説明を入力します。 |
   | プロジェクト | プロジェクト名を入力し、ドロップダウンリストから選択します。 |
   | 担当者（オプション） | 作業項目にユーザーを割り当てます。 |
   | 期日（オプション） | 期日を選択します。 |

1. 「**作成**」をクリックします。
