# ホームウィジェットフィルターの概要

以下のウィジェットのフィルターを使用して、作業の検索と整理を行うことができます。

* [マイプロジェクト](#my-projects)
* [マイタスク](#my-tasks)
* [マイイシュー](#my-issues)
* [マイリクエスト](#my-requests)
* [担当作業](#my-work)
* [マイ承認](#my-approvals)

>[!IMPORTANT]
>
>* ホームウィジェットにタスクと問題を表示するには、その親プロジェクトが現在のステータスまたは現在と同じステータスである必要があります。
>* また、プロジェクトは、現在ステータス、または表示するのに等しいステータスである必要があります。

## マイプロジェクト

[マイ プロジェクト] ウィジェットでは、次のフィルターを使用できます。

<table>
  <tr>
    <td>アクティブ</td>

  <tr>

    <td>「危険」ステータスのプロジェクトを表示 </td>
  </tr>
  <tr>
    <td>スケジュール遅れ</td>
    <td>「遅れているプロジェクト」ステータスを表示</td>
  </tr>
  <tr>
    <td>現在</td>
    <td>現在のステータスのプロジェクトを表示します。 </td>
  </tr>
  <tr>
    <td>期限が今月中</td>
    <td>現在のカレンダ月内にある完了予定日のプロジェクトを表示します</td>
  </tr>
  <tr>
    <td>遅延</td>
    <td>遅延ステータスのプロジェクトを表示</td>
  </tr>
  <tr>
    <td>マイプロジェクト</td>
    <td>自分に割り当てられたプロジェクト、または自分のプロジェクトチーム現在のステータスが表示されます。</td>
  </tr>
  <tr>

    <td>オン時間ステータスのプロジェクトを表示</td>
  </tr>
  <tr>
    <td>コスト予算超過</td>
    <td>実績コスト値が計画コスト値より大きいプロジェクトを表示します</td>
  </tr>
  <tr>

  <tr>

  </tr>
  <tr>

  </tr>
  <tr>
    <td>シナリオ プランナ プロジェクト</td>
    <td>シナリオ プランナで設定されたイニシアティブ ID を持つプロジェクトを表示します。</td>
  </tr>
</table>

## マイイシュー

[マイ リシュー] ウィジェットで次のフィルターを使用できます。

<table>
<tr>
    <td>すべて未割り当て</td>
    <td>割り当てられておらず、解決オブジェクトが添付されていないすべての問題を表示します。 </td>
  </tr>
  <tr>
    <td>自分に割り当てられている</td>
    <td>自分がメイン担当者である場合の課題を表示します</td>
  </tr>
  <tr>
    <td>完了</td>
    <td>課題を完了日とともに表示します </td>
  </tr>
  <tr>
    <td>自分が入力した</td>
    <td>入力した問題を表示します</td>
  </tr>
  <tr>
    <td>マイプロジェクト問題</td>
    <td><p>次の場所で問題を表示します。</p>
<ul>
  <li>私はプライマリ担当者です</li>
  <li>ステータスが不完全です</li>
  <li>解決オブジェクトがアタッチされていません</li>
</ul>
</td>
  </tr>
    <tr>
    <td>最近送信したマイ問題</td>
    <td><p>以下のような問題が表示されます。</p>
<ul>
  <li>私は提出しました</li>
  <li>解決オブジェクトがアタッチされていません</li>
  <li>ステータスが コンプリートプラン ではなく、完了日付がありません。</li>
  <li>今日の日付から3か月以内に提出された</li>
</ul>
</td>
  </tr>
    </tr>
    <tr>
    <td>自分で送信したイシュー</td>
    <td>自分が所有している課題を表示</td>
  </tr>
  <tr>
    <td>開く</td>
    <td>完了日付がない指摘事項を表示します。</td>
  </tr>
  <tr>
    <td>解決可能</td>
    <td>解決オブジェクトが添付されている問題を表示します</td>
  </tr>
  <tr>
    <td>マイ役割で未割り当て</td>
    <td>プライマリユーザーが割り当てられていないが、役割が割り当てられているすべての課題を表示します。 </td>
  </tr>
</table>

## 担当作業

[作業] ウィジェットでは、次のフィルターを使用できます。

<table>
  <tbody>
    <tr>
      <td>作業中</td>
      <td>現在作業中の項目を表示します</td>
    </tr>
    <tr>
      <td>開始準備完了</td>
      <td>以下を使用して項目を表示します。 
      <ul>
      <li>不完全な先行タスクやタスク制約はありません</li>
      <li>計画開始日付が過去または最大 2 週間後である</li>
      </ul>
      </td>
    </tr>
    <tr>
      <td>準備未完了</td>
      <td>を持つ項目を表示します
       <ul>
      <li>不完全な先行タスクまたはタスクの制約により、このアイテムは作業できません</li>
      または
      <li>予定開始日が 2 週間以上先の日付です</li>
      </ul>
       </td>
    </tr>
    <tr>
      <td>リクエスト日</td>
      <td>まだ作業を開始していない指摘事項が表示されます。</td>
    </tr>
    <tr>
      <td>私が委任済み</td>
      <td>他のユーザーに委任した項目を表示します</td>
    </tr>
    <tr>
      <td>私に委任済み</td>
      <td>ユーザーが委任したアイテムを表示します</td>
    </tr>
    <tr>
      <td>完了</td>
      <td>過去 2 週間以内に完了した作業が表示されます。 このフィルターオプションには承認は含まれません。</td>
    </tr>
  </tbody>
</table>
