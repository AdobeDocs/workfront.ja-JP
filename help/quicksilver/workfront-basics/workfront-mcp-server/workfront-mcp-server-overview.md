# Adobe Workfront MCP サーバーの概要

[!DNL Adobe Workfront] MCP （Model Context Protocol） サーバーを使用すると、サポートされているAI アシスタントで自然言語の会話を通じてWorkfront データを操作できます。 Workfrontをクリックしてプロジェクトを検索したり、タスクを更新したり、承認を確認したりする代わりに、AI アシスタントに平易な英語で質問し、AI アシスタントが作業を引き受けます。

Workfront MCP サーバーを使用するために、Workfront APIやMCPに関する知識は必要ありません。 AI アシスタントがWorkfrontに一度アクセスしてから、操作を開始するように設定できます。

## Workfront MCP サーバーを設定して使用する

Workfront MCP サーバーを使用する前に、Workfront管理者がWorkfrontでMCP サーバーアクセスを有効にする必要があり、AI アシスタントをWorkfrontに接続する必要があります。 詳しくは、[Adobe Workfront MCP サーバーの設定](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md)を参照してください。

設定が完了したら、AI アシスタントにWorkfront アイテムの検索、作成、更新、管理を依頼できます。 使用可能なアクション、リクエストの例、データ処理について詳しくは、[Adobe Workfront MCP サーバーの使用](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md)を参照してください。
