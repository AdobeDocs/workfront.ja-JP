# プログラムからプロジェクトを削除

次のいずれかを実行して、プログラムからプロジェクトを削除できます。

* 別のプログラムに移動します。
* 現在のプログラムから完全に削除し、別のプログラムに関連付けないでください。
* プロジェクトを完全に削除します。

  プロジェクトの削除について詳しくは、[プロジェクトの削除](../../../manage-work/projects/manage-projects/delete-projects.md)を参照してください。

## プログラムからプロジェクトを削除

1. プログラムに移動し、左側のパネルで「**[!UICONTROL プロジェクト]**」をクリックします。

1. 削除するプロジェクトを選択
1. 「**[!UICONTROL プロジェクトを削除]**」ドロップダウンメニューをクリックします。
1. **[!UICONTROL Delete]** > **[!UICONTROL Yes, Delete it]**&#x200B;をクリックして、Workfrontからプロジェクトを削除します。

   または

   「**[!UICONTROL プログラムから]**&#x200B;**[!UICONTROL 削除]**」をクリックして、選択したプログラムから削除します。

   「[!UICONTROL **プログラムから削除**]」をクリックした場合、プロジェクトは選択したプログラムと関連付けられなくなります。

   [!UICONTROL **削除**]&#x200B;をクリックすると、プロジェクトはWorkfrontから削除され、Workfront管理者はごみ箱から最大30日間復元できます。
