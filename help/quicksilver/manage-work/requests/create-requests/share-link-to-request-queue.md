# リクエストキューへのリンクの共有

## リクエストキューへのリンクの共有

1. リクエストの送信を開始します。詳しくは、 [Adobe Workfront要求の作成と送信](../../../manage-work/requests/create-requests/create-submit-requests.md).
1. 次のいずれかの操作を行います。

   * 「 Request Type 」フィールドでリクエストキューを選択します。
   * リクエストキューと最大 10 個のトピックグループを選択してください
   * リクエストキュー、トピックグループ、およびキュートピックを選択します

1. リクエストを送信する前に、 **共有パス** をクリックします。

   これにより、前の手順で選択したリクエストキュー、トピックグループまたはキュートピックへのリンクがクリップボードにコピーされます。

   <!--

1. 次のいずれかの操作を行います。

   * コピーしたリンクを他のユーザーと共有
   * ダッシュボードに外部ページとして埋め込む

      ダッシュボードへのリクエストキューの埋め込みについて詳しくは、 [リクエストキューをダッシュボードに埋め込む](../../../reports-and-dashboards/dashboards/creating-and-managing-dashboards/embed-request-queue-dashboard.md).
   共有リンクを使用すると、他のユーザーは、選択したリクエストキュー、トピックグループまたはキュートピックに直接移動できます。 リクエストが送信されず、ドラフトとして保存されている場合でも、この操作を実行できます。
