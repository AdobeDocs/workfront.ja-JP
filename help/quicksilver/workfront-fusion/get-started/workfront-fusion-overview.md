の役割 [!DNL Adobe Workfront Fusion] は、同じタスクを何度も繰り返す代わりに、新しいタスクに集中できるように、プロセスを自動化するためのものです。 アプリやサービス内でアクションをリンクして、データを自動的に転送および変換するシナリオを作成できます。 アプリまたはサービスのデータを監視し、そのデータを処理して目的の結果を提供するシナリオです。

シナリオは、アプリ内でのデータの変換方法や、アプリと Web サービスの間での転送方法を示す一連のモジュールで構成されます。

様々なアプリや Web サービスのモジュールを、タスクを実行するシナリオに配置します。 例えば、 [!DNL Workfront Fusion for Work Automation and Integration] ライセンスを使用すると、1 つのアプリまたは web サービスで新しいデータを監視するシナリオを作成し、そのデータをシナリオの後続のモジュールで使用して、新しいレコードの作成などの特定の結果を作成できます。 シナリオを設定したら、 [!DNL Workfront Fusion] タスクを自動的に実行します。

利用可能なについて詳しくは、以下を参照してください。 [!DNL Workfront Fusion] ライセンス， 「 [[!DNL Adobe Workfront Fusion] ライセンス](../../workfront-fusion/get-started/license-automation-vs-integration.md).

シナリオについて詳しくは、 [でのプラクティス統合シナリオの作成 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/get-started/create-a-practice-scenario.md).

モジュールについて詳しくは、 [モジュールのタイプ](../../workfront-fusion/modules/module-types.md).
