## [!UICONTROL Markdown からHTML]

このモジュールは、Markdown をHTMLに変換します。

このモジュールを設定する際には、次のフィールドが表示されます。

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>Markdown 形式のプレーンテキストを入力します。</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>GitHub Flavored Markdown を Markdown に変換する場合は、このオプションを有効にします。HTML</p> <p>詳しくは、[!DNL ]～のクダウンチートシート [!DNL GitHub] ドキュメント。</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!UICONTROL 不要部分を削除 ]</td> 
   <td>オプションを選択して、テキストからHTMLタグを取り除くか、エスケープHTMLを取り除くかを指定します。</td> 
  </tr> 
 </tbody> 
</table>

## [!UICONTROL Markdown へのHTML]

このモジュールは、HTMLコードを Markdown に変換します。

   <td> <p>このオプションを有効にして、HTMLを [!DNL GitHub Flavored Markdown].</p> <p>詳しくは、 [!DNL GitHub] ドキュメント。</p> </td> 
  </tr> 
 </tbody> 
</table>
