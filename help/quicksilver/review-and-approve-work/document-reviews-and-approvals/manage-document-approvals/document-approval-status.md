# ドキュメント決定ステータス

ドキュメントのステータスは、ドキュメントリストで直接確認できます。

![](assets/status-in-doc-list.png)

以下のステータスを使用できます。

<table>
            <col style="width: 50%;" />
            <col style="width: 50%;" />
            <tbody>
                 <tr>
                    <td>
                        Pending Review</p>
                    </td>
                    <td>
                        <ul>
                            <li>
                                レビュアーと承認者に通知しましたが、アセットはまだ開かれていません。
                            </li>
                        </ul>
                    </td>
                </tr>
                 <tr>
                    <td>
                        レビュー中</p>
                    </td>
                    <td>
                        <ul>
                            <li>
                                <p>少なくとも 1 人のレビュー担当者がアセットを表示しました</p>
                            </li>
                            <li>
                                <p>少なくとも 1 人のレビュー担当者がレビューを完了していません</p>
                            </li>
                            <li>
                                <p>このアセットに割り当てられている承認者はいません。</p>
                            </li>
                        </ul>
                    </td>
                </tr>
                 <tr>
                    <td>
                        レビュー済み</p>
                    </td>
                    <td>
                        <ul>
                            <li>
                                <p>すべてのレビュー担当者がレビューを完了しました</p>
                            </li>
                            <li>
                                <p>このアセットに割り当てられている承認者はいません。</p>
                            </li>
                        </ul>
                    </td>
                </tr>
                 <tr>
                    <td>作業が必要</p>
                    </td>
                    <td>
                        <ul>
                            <li>
                                <p>すべての承認とレビューが完了しました</p>
                            </li>
                            <li>
                                <p>「作業が必要」を 1 人以上の承認者が決定しました</p>
                                <p>他の承認者が、「変更を承認済み」または「承認済み」の決定を行った可能性があります
                            </li>
                        </ul>
                    </td>
                </tr>
                  <tr>
                    <td>変更後承認</p>
                    </td>
                    <td>
                        <ul>
                            <li>
                                <p>すべての承認とレビューが完了しました</p>
                            </li>
                            <li>
                                <p>少なくとも 1 人の承認者が「変更を承認済み」を決定しました</p>
                                <p>他の承認者が「承認済み」の決定を行った可能性があります
                            </li>
                        </ul>
                    </td>
                </tr>
                 <tr>
                    <td>承認済み</p>
                    </td>
                    <td>
                        <ul>
                            <li>
                                <p>すべての承認とレビューが完了しました</p>
                            </li>
                            <li>
                                <p>すべての承認者が「承認済み」の決定を行った可能性があります
                            </li>
                        </ul>
                    </td>
                </tr>
           </tbody>
        </table>
