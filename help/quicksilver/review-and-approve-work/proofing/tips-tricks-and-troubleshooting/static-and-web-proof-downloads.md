# インタラクティブなプルーフを新しいバージョンとして追加した場合、プルーフのバージョンをダウンロードまたはプレビューすることはできません

## 問題

ユーザーがインタラクティブなプルーフを新しいバージョンとして[静的プルーフ（.pdf、.png、.jpgなど） &#x200B;](/help/quicksilver/review-and-approve-work/proofing/proofing-overview/supported-proofing-file-types.md#static-files)に追加すると、以前のすべてのバージョンがダウンロードまたはプレビューできなくなります。

## ソリューション

プルーフビューアで静的バージョンを開き、サイドパネルの「**ダウンロード**」をクリックします。
