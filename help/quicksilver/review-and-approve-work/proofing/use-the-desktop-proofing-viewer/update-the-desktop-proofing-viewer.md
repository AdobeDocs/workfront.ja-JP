# デスクトップ校正ビューアを更新

デスクトップ校正ビューアで、改善点とバグ修正を含む新しいソフトウェアアップデートが利用できる場合があります。 ユーザーは、Adobe Workfront管理者の助けを借りずに、これらの更新を実行できます。

## 更新を手動で確認する

デスクトップ校正ビューアは、起動時に更新を自動的に確認します。 

更新を手動で確認するには：

1. Macで、 **Workfront Proof** > **Workfront Proof について** ( デスクトップ校正ビューアの上にあるAppleナビゲーションバー ) 

   または

   Windows の場合、 **ウィンドウ** > **Workfront Proof について** をクリックします。

   デスクトップ校正ビューアは更新を確認します。

1. 更新が利用可能な場合は、画面の指示に従ってデスクトップ校正ビューアを更新します。

## 標準アップデート

デスクトップ校正ビューアは、重要でない更新を行わずに機能できます。 ただし、最適な機能を実現するには、利用可能な場合に適用することをお勧めします。 次のページで利用可能になったら、通知が送信されます： **アプリの更新** ボタンを左サイドバーに置きます。

1. クリック **アプリを更新しました。**

   アップデートのダウンロード。

1. クリック **再起動します。**

   この **デスクトップ校正ビューア** 再起動し、配達確認の作業を再開できます。

## 必要な更新

デスクトップ校正ビューアに必要な更新を実行することが重要です。 その後、ビューアを使用し続けるには再起動する必要があります。

1. デスクトップ校正ビューアを開きます。
1. 表示されるボックスで、 **再起動します。**

   デスクトップ校正ビューアが再起動し、配達確認の作業を再開できます。
