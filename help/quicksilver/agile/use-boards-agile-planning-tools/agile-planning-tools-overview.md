# アジャイル計画ツールの概要

アドホックカードと接続カードを使用してボードを作成する場合、列 WIP 制限、スイムレーンへのカードのグループ化、バックログに使用できるインテーク列、ボードの集まりなど、追加のアジャイル計画ツールを活用できます。

これらのツールは、カンバンやスクラムなどの特定のアジャイルな手法にコミットすることなく、ユーザーやチームに適した方法で使用できます。
