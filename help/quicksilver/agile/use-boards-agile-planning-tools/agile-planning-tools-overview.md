# アジャイル計画ツールの概要

{{highlighted-preview}}

アドホックカードと接続カードを使用してボードを作成する場合、列の WIP 制限、スイムレーンへのカードのグループ化、バックログに使用できるインテーク列など、追加のアジャイル計画ツールを活用できます。 <span class="preview">板の集まり</span>.

これらのツールは、カンバンやスクラムなどの特定のアジャイルな手法にコミットすることなく、ユーザーやチームに適した方法で使用できます。
